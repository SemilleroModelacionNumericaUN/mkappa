# mkappa

Análisis no lineal de secciones transversales sometidas a flexión por el método de las fibras.

## Uso del programa

```
octave

>> MKAPPA [ADAT] [TLEC]
```

[ADAT] : nombre del archivo de entrada sin extensión

[TLEC] : identificador de la opción de lectura de datos
